Learning Management System

Gap Analysis
Conducted a detailed gap analysis to identify specific areas in the LMS that required improvement. This analysis highlighted functional discrepancies and alignment gaps, guiding targeted enhancements that closely aligned with both business objectives and user needs.
Target Operating Model (TOM)
...Developed a comprehensive Target Operating Model (TOM) to streamline workflows within the LMS, defining optimized processes to enhance efficiency and reduce redundancies. This TOM provided a strategic framework to guide system enhancements and ensure they supported broader organizational goals.
Unified Modeling Language (UML) Diagrams
Created UML diagrams to visualize and document complex workflows within the LMS. These diagrams clarified process structures, enabling a clear understanding of current and future states, which facilitated seamless collaboration with both technical and non-technical teams.
Collaboration with UX/UI Team
Worked closely with the UX/UI team to incorporate user-centered design principles into the LMS, improving overall usability and engagement. This collaboration focused on aligning system changes with user expectations, resulting in a more intuitive and satisfying experience for learners.
User Acceptance Testing (UAT)
Led User Acceptance Testing (UAT) sessions to validate system improvements before full deployment. These tests ensured that new features met user expectations and worked as intended, paving the way for a successful, user-approved LMS launch.
